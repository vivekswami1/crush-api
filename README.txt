# 💘 Crush API

A fun REST API that returns **compliments, flirty lines, romantic responses, and fun love calculations**.  
Built using **Spring Boot** and deployed on **Render**.

🌐 **Live API**  
https://crush-api-01ol.onrender.com

---

# 🚀 Features

- Random compliments
- Nerdy developer pickup lines
- Mood-based romantic replies
- Personalized messages
- Random reasons why someone is liked
- Fun love compatibility calculator
- Hidden Easter-egg endpoints

---

# 🛠 Tech Stack

- Java
- Spring Boot
- Maven
- Docker
- Render (Deployment)
